## 操作方法

| キー | 動作  |
|:--------:|:--------:|
| D | X軸+方向移動 |
| A | X軸-方向移動 |
| W | Y軸+方向移動 |
| S | Y軸-方向移動 |
| Q | Z軸+方向移動 |
| E | Z軸-方向移動 |
| NumPad6 | X軸+方向回転 |
| NumPad4 | X軸-方向回転 |
| NumPad2 | Y軸+方向回転 |
| NumPad8 | Y軸-方向回転 |
| NumPad7 | Z軸+方向回転 |
| NumPad9 | Z軸-方向回転 |
| LeftShift | 移動/回転速度上昇 |
| LeftCtrl | ワールド座標系で移動/回転 |
| Enter | 設定フォームを開く |
| R | リセット |
